# ♻️ AI Waste Segregation Guide

## SDG 12: Responsible Consumption and Production

### Project Overview
The AI Waste Segregation Guide is a Streamlit-based application that helps users classify waste into different categories for proper disposal and recycling.

This project supports Sustainable Development Goal (SDG) 12 by promoting responsible waste management and environmental sustainability.



## Problem Statement

Improper waste segregation leads to:
- Increased environmental pollution
- Reduced recycling efficiency
- Excess landfill waste

Many people are unaware of the correct category for different types of waste. This project helps users identify and segregate waste correctly.



## Solution

The application accepts waste items as input and classifies them into:

- 🟢 Wet Waste
- 🔵 Dry Waste
- 🔴 Hazardous Waste
- 💻 E-Waste

The system also provides disposal recommendations and highlights the environmental impact of proper waste segregation.



## Features

- Multiple waste item input
- Automatic waste classification
- Waste summary statistics
- Disposal guidance
- Environmental impact awareness
- SDG 12 aligned solution
